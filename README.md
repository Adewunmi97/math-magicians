## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- NodeJS - [v16.x](https://nodejs.org/en/)

### Setup

```bash
git clone https://github.com/Adewunmi97/math-magicians.git
cd ./math-magicians
```

### Install

```bash
npm install
```

### Usage

```bash
npm start
```

### Build

```bash
yarn build
```

### Run tests

```bash
npm test
```
